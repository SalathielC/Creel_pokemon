# 🔴 Pokémon Pack Counter 🔴

A stylish, easy-to-use web app to track how many Pokémon card packs you and your friends open in real life!

## Features

✨ **Beautiful Design** - Dark theme with Pokéball colors and smooth animations

📊 **Track Multiple Sets** - Create a counter for each Pokémon set you collect (Scarlet & Violet, Base Set, etc.)

📈 **Total Counter** - See your total packs opened across all sets at a glance

💾 **Auto-Save** - Your data is automatically saved to your device, so it persists between sessions

📱 **Fully Responsive** - Works perfectly on desktop, tablet, and mobile phones

## How to Use

### Online (Easiest for Friends)
1. Visit: `https://salathielc.github.io/Creel_pokemon/`
2. Start adding sets and opening packs!

### Offline (Local File)
1. Download or clone this repository
2. Open `index.html` in your web browser
3. Bookmark it for quick access

## How It Works

1. **Add a Set** - Type in a Pokémon set name (e.g., "Scarlet & Violet", "Base Set") and click "+ Add Set"
2. **Track Packs** - When you open a pack, click the ➕ button for that set
3. **View Progress** - See your total packs opened at the top and individual set counts below
4. **Manage Sets** - Use ➖ to decrement, "Remove Set" to delete a set, or "Reset All" to start fresh

## Sharing With Friends

You have a few options:

### Option 1: GitHub Pages (Recommended)
Friends can visit the link above and use it directly - no installation needed!

### Option 2: Share the File
- Send the `index.html` file to friends
- They can open it in any web browser
- Each person's device tracks their own packs

### Option 3: Local Network
- Run a simple local server and friends can connect on the same network
- (Ask if you need help setting this up)

## Technologies Used

- Pure HTML5
- CSS3 (with gradients and animations)
- Vanilla JavaScript
- Browser LocalStorage (for saving data)

No dependencies, no installation needed - just open and use!

## Notes

- Each device/browser has its own separate counter
- Data is stored locally on your device (not sent to any server)
- Clear your browser cache to reset everything
